# Notebook

Простой текстовый редактор на Python с использованием Tkinter.

---

## Требования

* Python 3.10+
* Библиотека `tkinter` (входит в стандартную библиотеку Python)

---

## Возможности

* Создание нового файла.
* Открытие существующих текстовых файлов.
* Сохранение изменений.
* Сохранение файла под новым именем.

---

## Структура проекта

* `main.py` — основной скрипт с графическим интерфейсом.
* Использует стандартные диалоги Tkinter для открытия и сохранения файлов.

---

## Использование

1. Клонируйте репозиторий:

```bash
git clone https://github.com/AKSLU/notebook.git
cd notebook
```

2. Запустите блокнот:

```bash
python main.py
```

3. В меню **Файл** доступны следующие команды:

* **Создать** — очистить текстовое поле для нового файла.
* **Открыть** — выбрать и открыть существующий текстовый файл.
* **Сохранить** — сохранить изменения в текущем файле.
* **Сохранить как** — сохранить текст в новый файл.

---

## Особенности

* Поддержка многострочного текста с переносом слов.
* Возможность отмены изменений (undo).
* Простое и интуитивное управление через меню.
